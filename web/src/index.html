表示するセグメント
Session
Idle timeout (min)
0 = never. AI sessions are terminated after this many minutes with no browser connected.
Hub crash grace (min)
0 = disable reconnect. How long wrappers wait for the Hub to come back after a crash before terminating their PTY. Applies to NEW sessions only — running sessions keep the value at the time they were spawned.
リモートアクセス保護
全アクセスを失効
Hub の token と認証 cookie を再生成し、紛失端末を含む全デバイスを弾きます。このPCも新しいURLで再ログインが必要になります。SSH/VPN の鍵失効は別途必要です（紛失時プレイブック参照）。
リモート PIN（任意）
既定 OFF。設定すると非 loopback（スマホ/VPN）からのアクセス時に PIN ログインを要求します。loopback（このPC）は従来どおり token のみで通ります。VPN 専用で SSH を置かない運用の追加の鍵向けです。

    認識エンジン
    ブラウザ内蔵認識のため、音声はブラウザベンダー（Google / Microsoft）の音声認識サーバへ送信されます。
    この端末ではブラウザ内蔵認識を利用できません。Whisper または OFF を選択してください。
    終了検知の待ち時間 秒
    無音で自動終了した後、直近の発話から指定秒数以内なら認識を継続します。0 で即終了（現状の挙動）。
    Whisper 接続 Hub 設定を使用
    Hub の config.yaml にある voice.whisper.server_url を使います。未設定の場合は録音後に設定エラーを表示します。

          About this app

          MANY-AI-CLI

          A tool to centrally manage approval operations and progress monitoring for multiple AI coding CLIs in one web UI. All processing is done locally.
